# Chapter 3: Cases

*Nominative, accusative, dative. Why German words change form — and why it's logical.*

---

## The Problem Cases Solve

English uses word order to show who's doing what:

```
The dog bites the man.
The man bites the dog.
```

Swap the positions and you swap the meaning. Word order carries the grammatical information.

German does it differently. German uses **cases** — the words themselves change form based on their role in the sentence. This means you can shuffle the word order freely and the meaning stays clear.

```
Der Hund beißt den Mann.    The dog bites the man.
Den Mann beißt der Hund.    The dog bites the man. (same meaning, different emphasis)
```

The articles changed: `der` → `den`. That change tells you who's doing the biting.

There are four cases in German. You need three of them immediately.

---

## The Nominative Case — The Subject

The **nominative** is the subject of the sentence: the person or thing doing the action.

Articles in nominative: **der, die, das, die** (same as what you learned in Chapter 2).

```
Der Mann schläft.       The man is sleeping.         (masculine subject)
Die Frau liest.         The woman is reading.         (feminine subject)
Das Kind spielt.        The child is playing.         (neuter subject)
Die Kinder spielen.     The children are playing.     (plural subject)
```

The nominative is the "dictionary form" — the form you look up.

---

## The Accusative Case — The Direct Object

The **accusative** is the direct object: the thing receiving the action.

For most genders, the accusative looks exactly like the nominative. Only **masculine** changes:

| | Nominative | Accusative |
|--|------------|------------|
| Masculine | **der** Mann | **den** Mann |
| Feminine | **die** Frau | **die** Frau |
| Neuter | **das** Kind | **das** Kind |
| Plural | **die** Männer | **die** Männer |

Indefinite articles:

| | Nominative | Accusative |
|--|------------|------------|
| Masculine | **ein** Mann | **einen** Mann |
| Feminine | **eine** Frau | **eine** Frau |
| Neuter | **ein** Kind | **ein** Kind |

Only masculine singular changes. Everything else stays the same.

```
Ich sehe den Mann.          I see the man.
Ich sehe die Frau.          I see the woman.
Ich sehe das Kind.          I see the child.

Ich kaufe einen Tisch.      I am buying a table.
Ich kaufe eine Lampe.       I am buying a lamp.
Ich kaufe ein Buch.         I am buying a book.
```

---

## The Dative Case — The Indirect Object

The **dative** is the indirect object: the person who *receives* the direct object, or who *benefits* from the action.

English uses "to" or "for" to signal this:

```
I give the book TO the man.
                    ↑ this is dative
```

Dative articles:

| | Nominative | Dative |
|--|------------|--------|
| Masculine | **der** | **dem** |
| Feminine | **die** | **der** |
| Neuter | **das** | **dem** |
| Plural | **die** | **den** (+ -n on noun) |

Indefinite dative:

| | Nominative | Dative |
|--|------------|--------|
| Masculine | **ein** | **einem** |
| Feminine | **eine** | **einer** |
| Neuter | **ein** | **einem** |

```
Ich gebe dem Mann das Buch.         I give the man the book.
Ich gebe der Frau das Buch.         I give the woman the book.
Ich gebe dem Kind das Buch.         I give the child the book.
Ich gebe den Kindern das Buch.      I give the children the book.
                                     (note: Kindern, not Kinder — plural dative adds -n)
```

---

## The Full Table

Learn this table. It's the core of German grammar.

|  | Masculine | Feminine | Neuter | Plural |
|--|-----------|----------|--------|--------|
| **Nominative** | der / ein | die / eine | das / ein | die / — |
| **Accusative** | den / einen | die / eine | das / ein | die / — |
| **Dative** | dem / einem | der / einer | dem / einem | den / — (+ noun -n) |

Pattern to notice: the dative masculine and neuter both use `dem`. The dative feminine uses `der` — same as nominative masculine. Confusing at first, logical once internalized.

A memory trick for dative: `dem, der, dem, den` — the "m" endings dominate, and feminine is the odd one out.

## Personal Pronouns in Cases

Pronouns change too:

| Person | Nominative | Accusative | Dative |
|--------|------------|------------|--------|
| I | **ich** | **mich** | **mir** |
| you (informal) | **du** | **dich** | **dir** |
| he | **er** | **ihn** | **ihm** |
| she | **sie** | **sie** | **ihr** |
| it | **es** | **es** | **ihm** |
| we | **wir** | **uns** | **uns** |
| you (plural) | **ihr** | **euch** | **euch** |
| they / you (formal) | **sie/Sie** | **sie/Sie** | **ihnen/Ihnen** |

```
Er sieht mich.          He sees me.        (mich = accusative)
Ich helfe dir.          I help you.        (dir = dative)
Sie gibt ihm das Buch.  She gives him the book. (ihm = dative)
```

---

## Which Verbs Take Which Case?

Most verbs take accusative for their object. A set of verbs always takes dative.

**Common dative verbs** (memorize these):

```
helfen      to help
geben       to give
sagen       to say / tell
zeigen      to show
schicken    to send
erklären    to explain
gefallen    to please / to like (special — see below)
gehören     to belong to
folgen      to follow
```

```
Ich helfe dem Mann.         I help the man.            (dem = dative)
Er sagt der Frau die Wahrheit. He tells the woman the truth.
Ich zeige dem Kind das Bild.   I show the child the picture.
```

**gefallen** — "to please" — works backwards from English:

```
Das Buch gefällt mir.       The book pleases me.   (= I like the book)
[subject] [verb] [dative]
```

To say "I like X" in German: `X gefällt mir`. The thing you like is the grammatical subject; you are the dative object. This is a common point of confusion.

```
Die Stadt gefällt mir sehr.     I really like the city.
Gefällt dir das Buch?           Do you like the book?
```

## Cases with Prepositions

Prepositions lock the case of the following noun.

**Always accusative:**
```
durch    through
für      for
gegen    against
ohne     without
um       around / at (time)
```

```
Das ist für den Mann.           That is for the man.
Ich gehe ohne einen Schirm.     I go without an umbrella.
Wir fahren durch die Stadt.     We drive through the city.
```

**Always dative:**
```
aus      from / out of
bei      at / near / with (at someone's place)
mit      with
nach     after / to (cities/countries)
seit     since / for (time)
von      from / of
zu       to
```

```
Ich komme aus der Schweiz.      I come from Switzerland.
Er wohnt bei seiner Mutter.     He lives with his mother.
Ich fahre mit dem Bus.          I go by bus.
Ich gehe zu dem Arzt.           I go to the doctor.
```

Note: `zu dem` contracts to `zum`, `zu der` to `zur`, `bei dem` to `beim`, etc.

```
Ich gehe zum Arzt.              I go to the doctor.
Ich gehe zur Schule.            I go to school.
```

---

## Two-Way Prepositions

Nine prepositions can take either accusative or dative, depending on meaning:

```
an      at / on (vertical surface)
auf     on / on top of
hinter  behind
in      in / into
neben   next to
über    above / over
unter   under / below
vor     in front of / before
zwischen between
```

**Accusative** = movement/direction (where are you going?):
```
Ich lege das Buch auf den Tisch.    I put the book onto the table.
Er geht in die Schule.              He goes into the school.
```

**Dative** = location/state (where are you?):
```
Das Buch liegt auf dem Tisch.       The book is on the table.
Er ist in der Schule.               He is in the school.
```

The trick: ask yourself — is something *moving to* a place (accusative), or *already at* a place (dative)?

```
Wo ist das Buch?  → auf dem Tisch (dative: location)
Wohin legst du es? → auf den Tisch (accusative: direction)
```

## The Fourth Case: Genitive (Brief)

There is a fourth case — **genitive** — for showing possession.

```
das Buch des Mannes     the book of the man / the man's book
die Tasche der Frau     the woman's bag
```

Genitive articles: **des** (m/n), **der** (f/pl)

Masculine and neuter nouns often add `-s` or `-es` in the genitive.

Modern spoken German increasingly uses `von` + dative instead:

```
das Buch des Mannes  =  das Buch von dem Mann
```

You'll encounter genitive in written German and formal contexts. For now, recognize it when you see it; use `von` when you need to express it.

---

## Summary: Article Changes by Case

The key changes to internalize:

- **Nominative**: the base form — der/die/das/die
- **Accusative**: only masculine changes — **den/einen**
- **Dative**: everything changes — **dem/der/dem/den**

A practical starting point: get accusative right (just masculine changes). Then learn dative as you encounter dative verbs and prepositions.

## Exercises

**1. Nominative or accusative?**

Fill in the correct article:

- ___ Mann schläft. (The man is sleeping — subject)
- Ich sehe ___ Mann. (I see the man — object)
- ___ Kind spielt. (The child is playing — subject)
- Wir kaufen ___ Tisch. (We buy the table — object)
- ___ Frau liest ___ Buch. (The woman reads the book)

**2. Dative with verbs**

Translate using dative:

- I help the woman. (helfen + dative)
- She gives the child the book.
- Do you like the city? (gefallen + mir/dir)
- He tells the man the truth. (die Wahrheit = the truth)

**3. Prepositions**

Fill in the correct article (accusative or dative):

- Das ist für ___ Frau. (für = always accusative)
- Ich fahre mit ___ Bus. (mit = always dative)
- Er geht in ___ Schule. (direction → accusative)
- Das Buch liegt in ___ Tasche. (location → dative)
- Sie kommt aus ___ Deutschland. (aus = always dative)

**4. Write it**

Write a paragraph (5–8 sentences) about going to a shop and buying something. Use at least:
- one nominative subject
- two accusative objects
- one dative (with `für` or `mit`)
- one two-way preposition

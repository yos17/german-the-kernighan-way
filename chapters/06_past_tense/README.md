# Chapter 6: Past Tense

*Two past tenses. When to use each. How to talk about what happened.*

---

## German Has Two Simple Past Tenses

English has one simple past: "I ate", "I went", "I saw".

German has two:

1. **Perfekt** — the past used in conversation: *"Ich habe gegessen"*
2. **Präteritum** (simple past) — the past used in writing: *"Ich aß"*

The meaning is the same. The choice depends on **context**, not time:

- Talking to someone? → Perfekt
- Writing a story or formal text? → Präteritum

With a few exceptions (`sein` and `haben` — always Präteritum in speech).

---

## The Perfekt (Conversational Past)

The Perfekt is formed with:

```
haben / sein  +  past participle (at end of clause)
```

```
Ich habe gegessen.          I ate / I have eaten.
Er hat geschlafen.          He slept.
Wir haben Deutsch gelernt.  We learned German.
Sie ist gefahren.           She drove / traveled.
```

**The past participle** is the key new form. Two types:

### Weak (regular) past participles

Pattern: `ge-` + stem + `-t`

```
machen → ge-mach-t    (made)
kaufen → ge-kauf-t    (bought)
lernen → ge-lern-t    (learned)
spielen → ge-spiel-t  (played)
wohnen → ge-wohn-t    (lived)
```

```
Ich habe Hausaufgaben gemacht.      I did homework.
Sie hat ein Buch gekauft.           She bought a book.
Wir haben Deutsch gelernt.          We learned German.
```

### Strong (irregular) past participles

Pattern: `ge-` + changed stem + `-en`

```
essen     → gegessen      (eaten)
trinken   → getrunken     (drunk)
schlafen  → geschlafen    (slept)
fahren    → gefahren      (driven/traveled)
gehen     → gegangen      (gone)
kommen    → gekommen      (come)
sehen     → gesehen       (seen)
lesen     → gelesen       (read)
schreiben → geschrieben   (written)
finden    → gefunden      (found)
nehmen    → genommen      (taken)
geben     → gegeben       (given)
stehen    → gestanden     (stood)
sprechen  → gesprochen    (spoken)
```

These must be learned individually. But patterns exist — many share the `-en` ending and similar vowel changes. They cluster around types (drink/think/sink all follow the same pattern in English too).

### Mixed (hybrid) past participles

A small group that adds `-t` like weak verbs but changes the vowel like strong verbs:

```
bringen → gebracht      (brought)
denken  → gedacht       (thought)
kennen  → gekannt       (known)
nennen  → genannt       (named)
wissen  → gewusst       (known/knowing)
```

## haben or sein?

Most verbs use `haben`. But **motion verbs and verbs of change of state** use `sein`.

**Use sein with:**

- Verbs of movement from place to place:
  ```
  fahren   → ist gefahren    (traveled)
  gehen    → ist gegangen    (went)
  kommen   → ist gekommen    (came)
  fliegen  → ist geflogen    (flew)
  laufen   → ist gelaufen    (ran)
  reisen   → ist gereist     (traveled)
  ```

- Verbs of change of state:
  ```
  aufwachen  → ist aufgewacht   (woke up)
  einschlafen → ist eingeschlafen (fell asleep)
  sterben    → ist gestorben    (died)
  werden     → ist geworden     (became)
  ```

- `sein` and `bleiben`:
  ```
  sein    → ist gewesen    (was/been)
  bleiben → ist geblieben  (stayed)
  ```

Rule of thumb: if the verb describes **movement** (and you could say "from A to B") or a **change** of state → use `sein`. Otherwise → use `haben`.

```
Ich bin nach Berlin gefahren.       I traveled to Berlin.
Ich habe das Auto gefahren.         I drove the car.
```

Subtle! When `fahren` means "to drive" (operating a vehicle, not going somewhere) → `haben`. When it means "to travel" (movement from A to B) → `sein`. In practice, this rarely trips people up.

## Perfekt Word Order

The auxiliary (`haben`/`sein`) goes in position 2. The participle goes to the end.

```
Ich    habe    gestern    ein Buch    gelesen.
[1]    [2]     [3]        [4]         [end]
subj   aux     time       object      participle
```

In a subordinate clause (verb goes last), auxiliary goes after the participle:

Wait — no. In subordinate clauses, the **finite verb** (the conjugated one) goes last. The finite verb here is the auxiliary:

```
...weil ich gestern ein Buch gelesen habe.
...because I read a book yesterday.
(participle before auxiliary in subordinate clause)
```

```
Er sagt, dass er gegessen hat.
He says that he has eaten.

Ich weiß, dass sie nach Berlin gefahren ist.
I know that she traveled to Berlin.
```

---

## Separable Verbs in Perfekt

The `ge-` goes between the prefix and the stem:

```
aufwachen → auf-ge-wacht     ist aufgewacht
einladen  → ein-ge-laden     hat eingeladen
anrufen   → an-ge-rufen      hat angerufen
einkaufen → ein-ge-kauft     hat eingekauft
aufstehen → auf-ge-standen   ist aufgestanden
```

```
Ich habe ihn angerufen.         I called him.
Sie ist früh aufgestanden.      She got up early.
```

## The Präteritum (Written Past)

For `sein` and `haben`, the Präteritum is used even in conversation — the Perfekt forms sound awkward:

```
war     (was)    not: ist gewesen
hatte   (had)    not: hat gehabt
```

| Person | sein (war) | haben (hatte) |
|--------|-----------|--------------|
| ich | war | hatte |
| du | warst | hattest |
| er/sie/es | war | hatte |
| wir | waren | hatten |
| ihr | wart | hattet |
| sie/Sie | waren | hatten |

```
Ich war gestern sehr müde.      I was very tired yesterday.
Er hatte keine Zeit.            He had no time.
Wir waren in Berlin.            We were in Berlin.
```

Modal verbs also use Präteritum in conversation:

```
konnte      could (past of können)
musste      had to (past of müssen)
durfte      was allowed (past of dürfen)
wollte      wanted (past of wollen)
sollte      was supposed to (past of sollen)
mochte      liked (past of mögen)
```

```
Ich konnte nicht schlafen.      I couldn't sleep.
Er musste früh aufstehen.       He had to get up early.
Sie wollte nicht kommen.        She didn't want to come.
```

## The Präteritum for Verbs (Written German)

In stories, novels, and formal writing, all verbs use Präteritum.

**Weak verbs**: stem + `-te` endings

| Person | Ending | kaufen → |
|--------|--------|----------|
| ich | -te | kaufte |
| du | -test | kauftest |
| er/sie/es | -te | kaufte |
| wir | -ten | kauften |
| ihr | -tet | kauftet |
| sie/Sie | -ten | kauften |

**Strong verbs**: vowel change + zero ending for ich/er

```
essen → aß       (ate)
fahren → fuhr    (drove/traveled)
gehen → ging     (went)
kommen → kam     (came)
sehen → sah      (saw)
lesen → las      (read)
schreiben → schrieb (wrote)
nehmen → nahm    (took)
geben → gab      (gave)
stehen → stand   (stood)
```

A narrative in Präteritum:

```
Es war ein kalter Winterabend.
It was a cold winter evening.

Ein Mann betrat das Café und setzte sich an einen Tisch.
A man entered the café and sat down at a table.

Er bestellte einen Kaffee und las die Zeitung.
He ordered a coffee and read the newspaper.

Niemand sprach mit ihm. Er sprach mit niemand.
Nobody spoke to him. He spoke to nobody.
```

---

## Which to Use?

| Situation | Tense |
|-----------|-------|
| Conversation, text messages, email | Perfekt |
| Stories, novels, newspaper articles | Präteritum |
| `sein`, `haben`, modal verbs (always) | Präteritum |

When in doubt: use Perfekt. You'll be understood.

---

## Summary

**Perfekt**: `haben/sein` + past participle

Participles:
- Weak: `ge-` + stem + `-t` (gemacht, gelernt)
- Strong: `ge-` + changed stem + `-en` (gegessen, geschrieben)
- Separable: prefix + `ge` + stem (aufgemacht, angerufen)

**sein** with: motion verbs (fahren, gehen, kommen, fliegen), change of state (aufwachen, sterben, werden), bleiben

**Präteritum**: used for `sein`/`haben`/modals in speech; for all verbs in writing

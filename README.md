# German: The Kernighan Way

*Learn grammar by using it. Every rule is a tool. Every chapter builds something real.*

---

German grammar has a reputation.

Four cases. Three genders. Adjective endings that change based on what's around them. Verbs that split in half and teleport to the end of sentences. Subordinate clauses where the verb goes last.

Most textbooks present these as mountains to memorize. This book presents them as a system — a logical, consistent set of rules that fit together once you see the shape.

You don't need to memorize everything at once. You need to use each piece until it becomes instinct. That's what this course is for.

## The Approach

Every chapter introduces one concept, explains *why* it works the way it does, and then puts it to use immediately in real sentences. The exercises aren't drills — they're small projects: write a paragraph, describe a scene, tell a story.

Grammar in isolation is dead. Grammar used is alive.

## How to Use This Course

Read the explanation. Then type the examples out loud. Then do the exercises.

You can't learn German grammar by reading about it. You learn it by producing it — making sentences, making mistakes, noticing patterns. The explanation gets you started. The practice makes it stick.

## Prerequisites

None. Start from zero.

If you know any German already, you'll move faster. But every chapter explains its topic from scratch.

## A Note on Pronunciation

This course focuses on grammar. Pronunciation is a separate skill — best learned from audio. Pair this with a pronunciation resource, a native speaker, or any app that lets you hear the language.
